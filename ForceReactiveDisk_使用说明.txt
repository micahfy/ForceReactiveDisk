ForceReactiveDisk 

====================
一、插件用途
====================
本插件用于管理力反馈盾牌(Force Reactive Disk)，在战斗中根据耐久与血量阈值自动切换盾牌，
并提供监控小窗显示盾牌状态，帮助玩家减少损耗、提升战斗稳定性。

====================
二、亮点功能
====================
1) 力反馈盾牌自动切换
   - 战斗中自动检测副手耐久，低于阈值时切换到背包中更高耐久的力反馈盾牌。
   - 所有力反馈盾牌都低于阈值时，进入紧急逻辑，避免过早损毁。

2) 勤俭盾牌模块(高级功能)
   - 可指定一块“勤俭盾牌”(非力反馈)作为低血量前的过渡盾牌。
   - 进入战斗后：血量高于阈值使用勤俭盾牌；血量降到阈值后切换到力反馈盾牌并保持到脱战。
   - 若所有力反馈盾牌耐久为 0，且勤俭盾牌在背包中，会强制切换到勤俭盾牌。

3) 战斗监控小窗
   - 显示副手与背包力反馈盾牌的耐久度。
   - 显示“勤俭盾牌”图标，可点击切换。

4) 小地图按钮
   - 左键打开设置界面
   - 右键启用/停用插件

5) 脱战自动装备银色黎明徽记
   - 脱战后自动将银色黎明徽记装备到饰品栏1。
   - 优先级：勇士印记(236351/236352) > 黎明符文/黎明之印(19812/13209) > 银色黎明委任徽章(12846)。

6) 脱战后自动恢复原饰品
   - 可设置延迟时间(30-180秒)，在未进战斗时自动把饰品恢复为原来的战斗饰品。
   - 短时间再次进战斗不会打乱记录，脱战后会继续尝试恢复。

7) 监控倒计时与快捷按钮
   - 监控小窗显示饰品恢复倒计时。
   - “+15秒”可延长本次倒计时。
   - “立即替换”可中断倒计时并立刻恢复原饰品(战斗中不可用)。

====================
三、安装方法
====================
1) 将插件文件夹放入:
   World of Warcraft\\Interface\\AddOns\\ForceReactiveDisk
2) 进入游戏，在角色选择界面启用插件。

====================
四、命令说明
====================
/frd 或 /frd check
  - 立即检测并按逻辑切换盾牌(可绑定宏)

/frd on
  - 启用插件

/frd off
  - 停用插件

/frd config
  - 打开设置界面

/frd reset
  - 重置监控与修理提醒位置

/frd status
  - 显示当前副手盾牌状态与背包力反馈盾牌数量

/frd monitor (mon)
  - 切换战斗耐久监控开/关

/frd monitor on/off
  - 显式打开/关闭监控

/frd monitor interval <秒>
  - 设置监控刷新频率(0.1-2.0)

/frd monitor ooc
  - 切换脱战显示监控

====================
五、设置说明
====================
1) 切换耐久度阈值
   - 低于该阈值时优先切换耐久更高的力反馈盾牌。

2) 主动检测模式
   - 勾选后，战斗中自动检测并切换盾牌。

3) 勤俭盾牌模块
   - 启用开关: 开启/关闭勤俭盾牌逻辑。
   - 血量阈值: 低于该阈值后将切换并锁定力反馈盾牌直到脱战。
   - 盾牌选择:
     * 拖拽背包盾牌到图标
     * 或点击图标使用当前副手
     * 右键清除设置

4) 检测频率/监控刷新
   - 检测频率: 自动切换逻辑的执行间隔
   - 监控刷新: 监控小窗刷新间隔

5) 其他
   - 脱战显示监控: 脱战时是否显示监控小窗
   - 修理提醒: 脱战后耐久过低时提示修理
   - 银色黎明徽记: 脱战后自动装备饰品栏1
   - 恢复原饰品: 脱战后延迟自动恢复饰品栏1

====================
六、常见问题
====================
1) 勤俭盾牌找不到
   - 插件会提示一次，并回退到力反馈盾牌逻辑。

2) 力反馈盾牌数量不足
   - 仅一块力反馈盾牌时，将只监控耐久，不做自动切换。

3) 频率太高导致卡顿
   - 建议检测频率 0.4-1.0 秒，监控刷新 0.4-0.8 秒。
